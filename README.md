## 介绍
Deepin Community Live CD 是一个让用户能够在系统出现问题时进行临时的维护和工作的镜像，预装了较为常用的维护工具，部分安装镜像还带系统安装功能。  
类似于 Windows PE 的东西  
Deepin Community Live CD QQ 交流群：881201853  
### 支持架构
amd64、arm64
